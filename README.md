# 🚀 C++ Programming Practice – 120 Programs

A collection of **120 C++ programming problems and solutions** completed as part of the **Summer Vacation Self-Learning Assignment** at **G.L. Bajaj Institute of Technology and Management**. The assignment covers fundamental to intermediate C++ programming concepts and is designed to strengthen problem-solving skills for coding interviews and placement preparation.

---

## 📚 Topics Covered

* 🔢 Basic Programs
* 🔁 Loops & Number Operations
* ⭐ Pattern Printing
* 🧮 Functions
* 📦 Arrays
* 🔍 Searching Algorithms
* 📊 Sorting Algorithms
* 🧩 Matrix Operations
* 📝 String Manipulation
* 🎯 Mini Projects
* 💡 Menu-Driven Programs
